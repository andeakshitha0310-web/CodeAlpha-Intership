# Music Generation with AI

This project uses Deep Learning (LSTM) to generate new music from MIDI datasets.

## Features

* MIDI file collection and preprocessing
* Note sequence extraction using Music21
* LSTM neural network training
* AI-generated music creation
* MIDI file export

## Technologies Used

* Python
* TensorFlow
* Music21
* NumPy

## Project Workflow

1. Collect MIDI files
2. Preprocess note sequences
3. Train LSTM model
4. Generate music
5. Save generated music as MIDI

## Output

Generated music is saved in:

generated_music/output.mid
